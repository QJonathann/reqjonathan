# **App Name**: qScheduler

## Core Features:

- Tutor Availability Management: Tutor can define their available tutoring hours, subjects offered, and rates via an intuitive calendar interface.
- Student Profile & Authentication: Students can register, log in securely, and manage their personal information and preferences.
- Session Browsing & Booking: Students can browse available tutors, filter by subject, view tutor availability in a calendar, and book specific time slots.
- Automated Lesson Reminders (AI Tool): An AI tool to craft personalized and engaging reminder messages sent to students and tutors before scheduled lessons, reducing no-shows and ensuring preparedness.
- Booking Confirmation & Notifications: After booking, students receive immediate confirmation emails/messages, and both parties are notified of booking status changes.
- Dashboard for Bookings: Separate dashboards for tutors to view their schedule and student details, and for students to track upcoming and past lessons.
- Basic Payment Gateway Integration: Students can make secure payments for booked sessions through a simple integration with a third-party payment provider.

## Style Guidelines:

- Color Scheme: Light scheme to maintain consistency with the existing qJonathan platform, emphasizing clarity and focus for an educational tool.
- Primary color: A vibrant blue (#2563EB) inspired by the existing brand, conveying trust, knowledge, and professionalism. This is the main interactive color for buttons and highlights.
- Background color: A very light, desaturated blue-gray (#EEF1F6) provides a clean, calm, and readable canvas, complementing the primary blue without distracting.
- Accent color: A deep cyan (#2989A9) is chosen for secondary interactive elements or status indicators, providing visual distinction while maintaining an analogous relationship with the primary blue.
- Headline and Body Font: 'Inter' (sans-serif) for its modern, clear, and highly readable characteristics, suitable for both large headlines and smaller body text within the scheduling interface.
- Icon Set: Continue using Lucide Icons for their clean lines, versatility, and professional appearance, consistent with the existing qJonathan website.
- Grid-based Layouts: Utilize a responsive grid system, similar to the main qJonathan site, for organizing content, forms, and calendar components efficiently across all device sizes. Focus on ample spacing and modular components for easy readability.
- Subtle UI Transitions: Incorporate subtle animations and hover effects, particularly for interactive elements like buttons, date selectors, and navigation, to provide user feedback without being distracting. (e.g., slight scaling, color changes, smooth fades).
